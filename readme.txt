Installation Used:
1. npm init
2. npm install express --save
3. npm install mongoose --save
4. npm install body-parser --save
5. npm install jsonwebtoken --save
6. npm install cors --save
7. npm install socket.io --save
8. npm install --save nodemailer
9. node index.js

Run the application (Node.js):
1. npm install
2. Run mongod.exe in shell (Windows - the directory of MongoDB installation),
    [create database (media-app), and add the collection (users), (news)] - not required
3. node index.js

Run the application (Angular):
1. npm install
2. ng serve
